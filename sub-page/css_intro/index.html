选择器权重 讲解

block：独占一行，宽度默认占满父元素，高度由内容决定，可以改变宽高
inline：不会自动换行，宽高由内容决定，不可改变宽高
inline-block：不会自动换行，宽高由内容决定，但是可以改变宽高
其它：block、inline、inline-block都有盒模型，但是inline元素不能用margin顶开其它元素

使用flex弹性盒子实现居中：将justify-content和align-items均设为center即可

使用position: absolute实现居中：top、left和transform: translate配合使用
translate语法：该函数有两个参数
        第一个参数：控制左右方向，为正往右边走，为负往左边走
        第二个参数：控制上下方向，为正往下边走，为负往上边走
        单位：绝对值px；相对值，以该元素的宽高为100%

使用margin实现居中
注意，margin: auto auto中的第一个auto（即上下方向）不起作用

文本居中：设置text-align、height和line-height

字体大小实际上指的是字体的高度，而字体行高（line-height）的设置如下
        normal：这种行高设置方式由浏览器和字体的大小共同决定
        1.5：相对值，行高为字体大小的1.5倍
        50px：绝对值，行高为50px
        200%：相对值，行高为字体大小的两倍
        5em：相对值，行高为字体大小的5倍
当选用行高时，通常使用数字，原因如下
        1、使用数字时，子元素的也会继承父元素的选择
        2、当使用其它相对值时，子元素继承的是父元素计算出来的行高
        3、也就是说对父元素是相对值，对子元素是绝对值
